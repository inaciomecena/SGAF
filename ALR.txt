sistema SaaS das Secretarias Municipais de Agricultura. O conteúdo inclui:

Arquitetura SaaS profissional (multi-tenant por município – código IBGE)

DER lógico com +40 tabelas

Documento de Requisitos Funcionais atualizado

A modelagem considera seu stack:

Backend: Node.js + Express.js

Frontend: React

Banco: MySQL

O sistema pode atender os 141 municípios de Mato Grosso e integrar dados estaduais da
Secretaria de Estado de Agricultura Familiar de Mato Grosso.

1. Arquitetura SaaS Profissional (Multi-Tenant)
Modelo de tenancy

Estratégia:

Single Database + Tenant Column

Tenant = codigo_ibge

municipio = tenant

Cada registro contém:

codigo_ibge
